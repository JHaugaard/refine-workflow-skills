## Key Decisions

### Planning Mindset Refinement (Session 2025-12-06)

**Discovery Protocol** (3 questions max):
1. **Scope**: "What's in scope, what's explicitly out?"
2. **Opportunity**: "What becomes possible for you by building this?" (journey-focused, not pain-points)
3. **Curiosity Prompt**: "What else should I know that I haven't asked?"

**Framing Exercise** (John's blend):
- Solution-Centric (primary): What gets built
- Outcome-Centric (secondary): What it enables
- Exploration-Centric (added): What you'll learn ("eager learner" identity)

**Approval Gates** (minimal friction):
- Two gates: Understanding ("Does this capture it?") + Handoff ("Ready to proceed?")
- Green/Yellow/Red signal language
- Silence never means proceed
- Dynamic "Light" mode - learning pauses decrease with experience

**Sequential Thinking**: Off by default, human-invoked only, rare
